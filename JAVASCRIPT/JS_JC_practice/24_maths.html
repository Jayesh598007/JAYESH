@ Unlike other objects, Math object dont need to create object first

@ syntax for Math properties : ' Math.property ' Math.E --> returns Euler's number Math.PI --> returns PI Math.SQRT2 --> returns the square root of 2 Math.SQRT1_2 --> returns the square root of 1/2 Math.LN2 --> returns the natural logarithm of 2 Math.LN10 --> returns the natural logarithm of 10 Math.LOG2E --> returns base 2 logarithm of E Math.LOG10E --> returns base 10 logarithm of E

@ syntax for Math methods : ' Math.method.(number) '
