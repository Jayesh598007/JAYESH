@ Variables defined with const cannot be Redeclared @ Variables defined with const cannot be Reassigned @ Variables defined with const have Block Scope @ As a general rule, Use const when you declare: >> A new Array >> A new Object >> A new Function >> A new RegExp

'const' variables must be assigned a value when declared

'const' variables cannot be reassigned

the keyword 'const' here defines constant referance to a value Because of this you can NOT: >> Reassign a constant value >> Reassign a constant array >> Reassign a constant object But you CAN: >> Change the elements of constant array >> Change the properties of constant object

The x declared in the block, is not same as x declared outside the block

Redeclaring an existing var or let variable to const, is not allowed

reassigning a variable with const, in same scope, is not allowed

redeclaring a variable with const, in another scope/ block, is allowed
