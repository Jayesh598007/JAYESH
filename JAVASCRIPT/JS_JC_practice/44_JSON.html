@ JSON is a format for storing and transporting data @ JSON is often used when data is sent from a server to a web page

>> JSON stands for JavaScript Object Notation >> JSON is a lightweight data interchange format >> JSON is language independent(JSON data can be written in any programming language) >> JSON is easy to understand Syntax: { "objectName": { "name": "Jayesh", "age": 23, "car": "volvo", "dept": "Dev-ops" } } # Syntax Rules: >> data is in form of key:value pair >> keys are quoted >> values are quoted (if req) >> data is separated in columns
