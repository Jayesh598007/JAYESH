@ Regular expression is a sequence of characters that forms a search pattern Syntax: /pattern/modifiers; Example: /hello/i; >> 'hello' is a pattern (to be searched) >> 'i' is a modifier( for case-insensitive)

search() method searches a string for a specified value and returns the position of the match

replace() method replaces a specified value with another value in a string

@ Regular Expressions Modifiers: i >> Perform case-insensitive matching g >> Perform a global match (find all matches rather than stopping after the first match) m >> Perform multiline matching @ Regular Expressions patterns: [abc] >> Find any of the characters between the brackets [0-9] >> Find any of the digits between the brackets (x|y) >> Find any of the alternatives separated with @ Quantifiers: n+ >> Matches any string that contains at least one n n* >> Matches any string that contains zero or more occurrences of n n? >> Matches any string that contains zero or one occurrences of n

test() method searches a string for a pattern, and return 'true' or 'false'

exec() method searches a string for a pattern, and return found text
