@ Variables defined with 'let' cannot be Redeclared @ Variables defined with 'let' have Block Scope, thus limited to the function { } only

@ Variables defined with 'var' can be Redeclared @ variables defined with 'var' have global scope, even if defined in { }, can be accessed outside the block
