'+' operator is used to add(concatenate) two strings

'+=' operator is also used to add two strings

Operator precedence describes the order in which operations are performed

( ) // Expression grouping // (3 + 4) . // Member // person.name [] // Member person // ["name"] () // Function call // myFunction() new // Create // new Date() | // Bitwise OR // x | y & // Bitwise AND // x & y ^ // Bitwise XOR // x ^ y << // Shift left // x << 2 >> // Shift right // x >> 2
