@ Loops can execute a block of code a number of times Kind of Loops: for - loops through a block of code a number of times for/in - loops through the properties of an object for/of - loops through the values of an iterable object while - loops through a block of code while a specified condition is true do/while - also loops through a block of code while a specified condition is true

@ For - loops through a block of code a number of times Syntax: for (statement 1; statement 2; statement 3) { // code block to be executed } statement 1 is executed (one time) before code block execution statement 2 defines condition for code block statement 3 is executed (every time) after code block executed

@ For in - loops through the 'properties' of an Object Syntax : for (key in object) { // code block to be executed }

@ For of - statement loops through the 'values' of an iterable object Snytax: for (variable of iterable) { // code block to be executed }

@ while - loop loops through a block of code as long as a specified condition is true Syntax: while (condition) { // code block to be executed }

@ do while - loop will always be executed at least once, even if the condition is false, because the code block is executed before the condition is tested Syntax: do { // code block to be executed } while (condition);
